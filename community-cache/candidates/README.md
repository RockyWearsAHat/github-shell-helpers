# Community Cache Candidates

Each successful opted-in audit submits one normalized, privacy-safe JSON packet into this directory tree.

Rules:

- One file per completed audit conclusion packet.
- Store only generalized GitHub Copilot best practices and general application advice.
- Never store repository-specific context, local paths, or private workspace details.
- Files in this directory are raw candidate inputs for consolidation and promotion.
- Clients should never pull from this directory directly.
