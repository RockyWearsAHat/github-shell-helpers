## Release Scope

Finishes the Node-free migration: the dead Node runtime is **deleted**, and every
package channel installs the prebuilt native binary via a shared bootstrap. Also
removes the `--course` option from grading (the course is auto-detected).

## Removed (dead Node runtime)

- `helpers` (Node CLI — now the native binary), `helpers-server[.js]`,
  `helpers-serverd.js` (daemon), `helpers-mcp.c` (C shim), `git-research-mcp[.js]`,
  and all `lib/mcp-*.js` (the Node MCP/search implementation — replaced by Rust).
- The matching Node tests and the `git-research-mcp` man page.

## Node-free packaging

- A shared `scripts/fetch-prebuilt.sh` bootstrap downloads the prebuilt
  `helpers-native` for the host, symlinks the `helpers`/`git-*` CLIs, and registers.
- `.deb`/`.pkg` postinstalls and the portable tarball use it; the npm package is now
  a thin shim that downloads the binary on install (`scripts/npm-postinstall.cjs`).
  No package ships or runs Node for the helpers runtime.
- The VS Code extension launches `helpers-native mcp` (no Node server).

## grade

`helpers grade <path>` no longer takes `--course` — the course (CS2420 vs CS3500) is
auto-detected from the project. `--json` is still supported.
