## Release Scope

The npm package is now scoped as **`@rockywearsahat/helpers`** (cleaner name than
`github-shell-helpers`). It's a thin shim that downloads the prebuilt native binary
on install. Release wiring is now scope-agnostic.

## Changes

- npm package renamed `github-shell-helpers` → `@rockywearsahat/helpers`
  (`npm i -g @rockywearsahat/helpers`).
- Release upload/attach/publish steps match the package tarball by glob, so the
  scoped name flows through automatically.

## Coming next (other package managers)

Homebrew/Scoop/Winget/apt all consume the per-platform prebuilt binary + its
checksum, so they'll be generated in the release-publish job (where those are
available); Winget/Scoop also need a `.zip` artifact and apt needs a GPG signing
key + a hosted repo. Tracked as the next step.
